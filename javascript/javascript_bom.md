# The Browser Object Model (BOM)

BOM это объектная модель браузера в которую входят следующие функции:  
- navigator
- screen
- location
- frames
- history
- alert
- confirm
- prompt
- XMLHttpRequest

```
alert( location.href ); // выведет текущий адрес
```
